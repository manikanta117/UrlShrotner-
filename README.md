# URL Shortener 🚀

A simple URL Shortener backend project built using Java and Spring Boot.

## Features
- Shorten long URLs
- Redirect short URLs to original URLs
- REST API support
- MySQL database integration

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
